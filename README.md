# Crypto Telegram Bot 🤖

A comprehensive cryptocurrency Telegram bot with multiple features including price checking, trending coins, market data, and token safety checks.

## Features 🚀
- Real-time cryptocurrency prices
- Trending coins list
- Global market data
- Token safety checks (rug check)
- Search coins by name or symbol
- DeFi data integration
- Inline keyboard navigation
